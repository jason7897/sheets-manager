# 개발 로그 — Sandwich House

**날짜**: 2026-04-06  
**프로젝트**: Sandwich House (샌드위치 레시피 홈페이지)

---

## 프로젝트 개요

집에서 쉽게 만들 수 있는 샌드위치 레시피를 소개하는 정적 웹페이지.  
외부 라이브러리 없이 **HTML5 / CSS3 / Vanilla JS** 만으로 구현.

## 2차 작업 — 검색·필터·즐겨찾기 추가

### 변경 파일
- `index.html` — 툴바 섹션 추가, 카드에 `data-id` / `data-category` 속성 추가, 즐겨찾기 버튼 추가
- `style.css` — 툴바·즐겨찾기 버튼·empty-state 스타일 추가
- `script.js` — 신규 생성

### 구현 기능

#### 실시간 검색
- `input` 이벤트로 타이핑할 때마다 필터 즉시 적용
- 카드의 전체 `textContent`를 대상으로 검색 (이름·설명·재료·조리법 모두 포함)
- 검색어 입력 시 ✕ 초기화 버튼 표시

#### 카테고리 필터
- 필터 버튼 6종: 전체 / 클래식 / 채식 가능 / 채식 / 프리미엄 / 건강식
- 카드의 `data-category` 속성과 매칭
- `active` CSS 클래스로 선택 상태 표시

#### 즐겨찾기
- 각 카드 우상단 ♡ 버튼으로 개별 등록/해제
- 등록 시 ♥ + 핑크 배경으로 상태 변경, scale 애니메이션 (1.4배, 250ms)
- `localStorage` 에 `Set → JSON` 형태로 영속 저장 (새로고침 후에도 유지)
- "♡ 즐겨찾기" 필터 버튼으로 즐겨찾기 항목만 보기

#### 복합 필터 로직

```
보이는 카드 = 카테고리 일치 AND 검색어 포함 AND (즐겨찾기 필터 ON이면 즐겨찾기 여부)
```

세 조건을 `applyFilters()` 함수 하나에서 통합 처리.

#### 빈 결과 처리
- 조건에 맞는 카드가 없으면 안내 메시지 + "필터 초기화" 버튼 표시
- 초기화 버튼 클릭 시 검색어·카테고리·즐겨찾기 필터 모두 리셋

### 주요 기술 결정

| 결정 | 이유 |
|------|------|
| Vanilla JS | 외부 의존성 없이 가볍게 유지 |
| `data-*` 속성 기반 필터 | JS 데이터 배열 불필요, DOM만으로 처리 |
| `localStorage` 즐겨찾기 | 서버 없이 영속화, 페이지 새로고침 대응 |
| 단일 `applyFilters()` 함수 | 검색·카테고리·즐겨찾기 복합 필터를 한 곳에서 관리 |
